## 📝 **Data Entry Steps:**

1. **Login to Admin Panel:** `/admin/products`
2. **Click "Create New Product"**
3. **Select appropriate category**
4. **Fill in product details** from above guide
5. **Add product images** (if available)
6. **Set "Is External" to No** for all products
7. **Leave "External URL" empty** for all products
8. **Add features** using the features array
9. **Set pricing info** to "Contact for pricing and demo"
10. **Save each product**

---

## 🎯 **Important Notes:**

- **All products are INTERNAL** - No external redirects
- **"Contact for Details" approach** - All products lead to enquiry forms
- **Features array** - Use the new features field for key features
- **Pricing strategy** - "Contact for pricing" for all products
- **Images** - Add product images where available
- **Descriptions** - Detailed descriptions for SEO and user understanding
